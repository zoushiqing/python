萧添寿质问孙阳，人们都知道杜阿岛海峡那个地方并不是非常适合潜水，你为什么还要带她去？而且，你为什么不下海？却让陈洁妮下海？ 孙阳回答，因为上次去那里潜过水的时候，我们并没有遇见危险。而且并不是我不潜水，而是我得讲究礼貌，女士优先，您懂吧？ 萧添寿再次质问孙阳，当你发现陈洁妮消失的时候，你为什么不立即下海找她？你告诉约瑟夫你不会游泳，可实际上你是个游泳高手。好，就说你当时被吓到了，不知道怎么办。可到后来，当五位渔民下海找陈洁妮的时候，你还是站在船上，不下去潜水，啥也不做，连脚都是干的，麻烦你来解释一下，为什么你不下海寻找陈洁妮？ 孙阳说，是这样，她是我的未婚妻，我很在乎她，肯定想找到她。但是我一直在观察海面，根本没有任何气泡从海底冒出来，所以你明白吧，她这肯定是死了。死了就是死了，我能怎么办？那么大的海，我上哪去找她？ 萧添寿的最后一次质问，是关于保险的。萧添寿说，我们来聊聊保险的事情吧，陈洁妮身上意外险的保险额度达到40万新币（约200万元人民币），而她只是一个酒吧服务员，一个月只能挣300多块新币（约1700元人民币），她还有两个孩子要养，她为什么要花这么多钱给自己买保险？
